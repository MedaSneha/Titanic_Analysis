# Titanic_Analysis
## Project Objective

“The objective of this dashboard is to analyze passenger survival patterns in the Titanic dataset based on gender, age, passenger class, embarked location, and fare paid.”

## Questions(KPIs)

- How many passengers were on the Titanic?
- What percentage of passengers survived?
- Did females survive more than males?
- Which passenger class had the highest passengers?
- Which age group had the highest passengers?
- From which port did most passengers embark?
- Did passenger class affect survival chances?

## Process
- Collected and imported the Titanic dataset into Power BI for analysis.
- Performed data cleaning and transformation by handling missing values and creating useful              measures/KPIs.
- Built interactive visualizations such as KPI cards, donut charts, bar charts, and gauge charts.
- Analyzed passenger survival patterns based on gender, age, passenger class, fare, and embarked         location.
